# MetaTube Inspector

A modern and professional starter template built with **Next.js** and **Firebase Studio**, designed for performance, scalability, and developer experience.

## 🚀 Tech Stack

- **Next.js 14+ (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **Firebase Hosting**
- **Vercel (CI/CD & Hosting Preview)**
